ChattyChimpChat

ChattyChimpChat is a chattier version of the ChimpChat implementation used in the Android SDK.

Overview

This is a chattier chimpchat implementation based on this commit from the Android SDK.

It will throw most exceptions occurring in the code thus enabling you to catch them! Please download the whole repository.

Development

Have a look at the Github repository. This version does not have the hierarchyviewer.
